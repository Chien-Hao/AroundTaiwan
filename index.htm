【環島一日遊全記錄】- 建豪 2007/07/30

 

【緣起】

　　話說一兩年前，偶然之間看到之前系上一位楊教授的個人網頁，內有他個人的小品文章，有一篇是提到教授個人曾於2003年9月完成此一壯舉，全文在此。不知道大家看完文章之後的感想是怎樣？有人會覺得這樣太累了；有人會覺得這根本就是瘋子；也有人會覺得這簡直只是浪費時間而已，一點意義都沒有。

　　但是，也許是射手座的體內真的都有一股勇於冒險的基因吧，我看完網頁之後的感想是，哇！好讚阿！有機會我也要嘗試一下！於是乎，一顆瘋狂的種子，就這麼悄悄的植入我的心中。

 

【前言】

　　但是想歸想，一直住在家裡的我，連平時都有門禁了(你沒看錯，身為一個大男生的我，居然有門禁，時間是ＰＭ10:30)，更別說要出門24小時不回家睡覺，那不是東方夜譚也不是西方夜譚，根本就是天方夜譚！！也因此，這顆瘋狂的種子，雖已經植入心中，但是卻永遠長不大阿！

　　然而，自從去年九月開始上台北唸書之後，似乎任何瘋狂的種子都可以開始灌溉施肥了。 不過，平日有跟在中研院當助研究員的學長一起做研究，所以平常日都要去中研院，沒去中研院的日子也要家教，假日不是跟朋友出去玩，就想要休息，要挑一個合適的時間出發，也有一定的困難度。

　　但是，這一兩個禮拜學長剛好出國，讓我可以偷閒一下，再加上上個禮拜一個大學同學坐火車環島一圈，對我來說，就像是將這顆種子灌上生長激素一樣，一下子長的跟大樹一樣高了！而且，最近不是有一句話很有名嗎？ 「有些事，現在不做，一輩子都不會做了！」可不是嗎？一日環島就跟談戀愛一樣，是需要一些勇氣和衝動的！更何況，現在的我，沒有工作、沒有家庭、沒有女友、沒有爸媽的掌控，是我這輩子最自由的時候，不待這時創造些屬於自己的回憶，更待何時？於是乎，挑個兩天完全沒事的日子，計劃安排一下，就出發了～

　　也許有人會問，為什麼不騎腳踏車或是騎機車環島呢？不是更具挑戰性嗎？騎機車我是有想過，但是我現在那輛老爺車已經跑七萬多公里了，不要說跑一圈了，我看連1/4圈都有問題！到時候要是在人跡罕至的地方半路拋錨，那就欲哭無淚了。而騎腳踏車，少說也要一個禮拜十天的，現在要抽出一個禮拜的空，根本就不可能！因此權衡之下，還是開車比較舒適也可以順便練練開車技術。

【前一天晚上】

　　 在食衣住行育樂這些方面，其實要安排的只有"行"而已，參考一下教授的行程，打開電子地圖，因為想要在東部三縣市 ( 宜蘭、花蓮、台東 ) 還有南部兩縣市 ( 高雄、屏東 ) 的火車站合照，所以預計行程如下：宿舍 → 租車行 → 辛亥路 → 國道3甲 → 國道五號 → 雪山隧道 → 宜蘭火車站 → 七星潭 → 花蓮火車站 → 三仙台 → 台東火車站 → 屏東楓港 → 屏東火車站 → 高雄火車站 → 中山高速公路 → 建國高架橋→ 租車行→ 宿舍。

　　在交通工具方面，也打電話去租車行，詢問一下相關的細節，只要明天早上再去領錢，就一切準備完畢囉！為了要儲備充分的體力，前一天晚上也提早就寢。

 

【正式出發】

　　就跟小時候要去遠足的前一天一樣，前一天晚上肯定是興奮到輾轉難眠的阿！沒錯，我前一天也沒睡好，在床上翻來覆去的，最後"應該"是有睡著，不過七點就醒了，比預定時間早一個多小時。盥洗完畢吃完早餐原本想回去再躺一下，無奈還是睡不著，便作罷。

早上再打一通電話跟車行老闆約好九點半去取車，結果我九點十分不到就到車行了，等了將近一小時老闆才來開門，期間還有打電話給老闆催他，我想，客人還要催老闆開門，這應該是很少見的吧？取車時，跟老闆凹說，讓我等這麼久，能不能少算個100，但是老闆不給我凹，沒輒！被老闆這麼一拖，等我租到車加完油，要準備出發，時間已經是10:38囉，anyway，環島一日遊，正式出發！！

 

　　車行其實離辛亥路不遠，到了辛亥路再接國道三甲，再接雪山隧道的國道五號，因為非假日的關係，一路上都很順，一小時左右就到宜蘭火車站了，原本要找宜蘭的朋友順便吃午餐，結果他人現在不在宜蘭，只好自己隨便找一間店就吃囉。

　　接著，走蘇花公路往花蓮，今天天氣很好，沿途的風景也不錯，因此，看到喜歡的景點，我就停下車拍拍照。往花蓮的途中，看到一個客來堡休息站，解決一下生理需求(上洗手間，請不要想歪)，再照照風景，便繼續上路。沒多久看到太魯閣國家公園的指標，原本這不是在我的行程中的，沒想到順路會經過，當然是要去照相留念一下，來到太魯閣想說那我順便去天祥照個相，再折返好了(怎麼跟教授的行程一樣？好啦，我承認我是參考教授的行程)，教授在天祥這邊是因為塞車只好中途折返。結果我去的時候，一路順暢，沒錯，應該是因為非假日關係。正在竊喜我的行程比較順利的時候，俗話說的好，計劃永遠趕不上變化，代誌也絕對不是憨人所想的這麼簡單，開到一半，就被堵住了，原因是前面在修路，每隔整點才放行10分鐘，當時時間是15:44，離整點時間不多，那就等吧！放行之後，雙方在會車時，我突然想到，等我回程時，不是又要整點才能放行了？！聰明的我當機立斷，馬上折返，加入對向車道的行列中，換句話說，天祥這景點，我也失敗了！

　　下一個點是七星潭( 怎麼又跟教授一樣？巧合巧合)，在這邊算是稍作休息，看看海，吹吹海風，一個人也是挺愜意的！有一個在花蓮當兵的朋友剛好今天要收假，原本說好五點可以到車站，結果因故要六點才到，就在七星潭這邊多待一點時間囉。接著，往花蓮火車站的途中經過七星柴魚博物館，照個相，是一定要的啦！在火車站接過朋友之後，朋友提議要去公正街一間有名的包子店吃包子當晚餐，結果，包子好吃是沒錯，但是我們的小籠包等了超久才來，小不爽，朋友還差一點逾假未歸。跟朋友閒聊才發現，我中午在宜蘭吃的，結果晚餐在花蓮吃，這....我島還環的完嗎？？！看樣子好戲還在後頭阿！後面的行程可刺激了！送完朋友去營區之後，因為行程delay，便開始趕路了。趕路不久就看到指標，往台東還有166KM，嗯，很好，就把倒數里程當作樂趣吧，這樣想就不覺得很遠了。

　　在到台東火車站之前的預計行程是三仙台， 而三仙台也算是一個知名景點，因此沿路我都在注意往三仙台的指標，結果沒看到三仙台的指標，反而是看到台東火車站的指標，印象中是會先經過三仙台才到火車站的。問當地人的結果是，三仙台在海線 (台11線) 而我開的是山線 (台9線)，台9線有一個岔路可以彎到台11線，但是指標是寫 "東部海岸"，我不知道可以通往三仙台，就錯過了。而現在從火車站再返回三仙台的話，要一個小時，只好放棄了。到台東火車站，時間已經是22:42了。

　　接著是要往屏東，一樣，很快就看到指標，寫著163KM到屏東。剛剛166KM都開完了，163KM算什麼？走的路線是連接台東和屏東的南迴公路，這邊是我預計最危險的地段，一方面時間是晚上，視線比較不好，也接近睡覺時間，怕會想睡覺，一方面又是山路，要是一個不注意，各位就不是在電腦前看我的遊記，而是在電視上看我的新聞了。結果意外的，我的精神很好，可能是跟我三不五時會鬼叫一下提振精神有關，一路上也很順暢，深夜時段，車不多，倒是中途有一個檢查站，被攔下檢查駕照。接著台九線結束接台1線，時間已經是 00:45了， 行程也走到後半段了，接下來要做的是就像是周杰倫的一首歌－「一路向北」。

　　到達屏東火車站的時間是01:48，突然覺得深夜到達也有個好處，拍照的時候都沒有人，比較不會被當作怪叔叔看待。往高雄火車站的路上，一時找不到台1線，還問了一下當地人。到了高雄火車站，照完相，剛好警察伯伯來巡邏箱簽名，印象中是要去九如路才有交流道，但是想說建國路比較順，不用再轉到九如路，便問一下警察伯伯 (我絕對不會跟警察伯伯說我是高雄人的)，果然要去九如路才可以上高速公路，警察伯伯還叫我開車要小心，果然人民的褓母，真是貼心。

　　上交流道時，時間已經將近凌晨三點了，中山高這邊預計的行程就是每個休息站都去拍照留念，由南而北依序是：仁德、新營、西螺、泰安、湖口、中壢。正所謂，「好酒沉甕底」，行程到這邊，是最刺激、最有挑戰性的，因為已經到了睡覺時間了，就算是平常熬夜，最晚也三點多睡覺，而我又想要挑戰一下，堅持不休息，連小睡半小時都不肯，一方面是怕一睡不起，要多付一天租車錢，一方面是有股莫名的堅持，要維持本次旅程的高度挑戰性。因此，到後面的休息站時，我都要用水洗個臉清醒一下自己，提振一下精神，不過通常只有上路的前十分鐘有效，後來上下眼皮就不聽使喚，想要相親相愛在一起了。當時時速100上下，有些瞬間會恍一下神，車子就會晃動一下，好啦，我承認，這真是有點危險啦，不過叔叔是有練過的，小朋友不要學喔！而在這邊，也從天暗慢慢開到天亮了。

　　最後，如同之前規劃的，從圓山交流道下高速公路，接建國高架橋，再到羅斯福路的租車行，結束了這次的環島一日遊，時間是08: 03。不過回到租車行時，老闆還沒開門，只好把車大搖大擺的停在車行門口，不熄火吹冷氣，補眠一下了，睡到快十點老闆才來開門。老闆還很懷疑我真的環島結束了嗎？還是只是開去墾丁來回而已。看樣子，這篇遊記應該給他看看的！

　　共計此趟環島，總時間是21時25分，總里程數是 985.7 KM。

 

【心得感想】

　　 誠如教授所說的，要完成這件事其實沒有什麼，一般人只要會開車，加上一點點的衝動就可以辦到，但是，我想說的是，應該是很大的衝動，才會去做，因為一般人光是聽到一天繞台灣一圈就累了，更別說要去做。而我，也終於完成了我的一樁心願，不然一顆心一直懸在那邊，每次想要做，都會找理由不做，再拖下去，可能就真的一輩子都沒機會了。創造了一個這樣不同的回憶，除了照片，當然要用點文字紀錄一下，不但是記錄著自己的足跡、自己的回憶、自己的想法，更是紀錄自己的生命。什麼？你問我以後會不會再做一次？呵呵，要環島當然會，但是到時肯定是帶著自己心愛的女友(老婆)，環島一週慢慢玩，創造一些屬於我們的浪漫回憶，這樣才更有意義，各位看倌，您說是嗎？
